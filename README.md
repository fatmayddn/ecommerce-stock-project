# E-Commerce Stock Automation

## Proje Açıklaması
Bu proje, e-ticaret ürün verilerini API ve CSV dosyasından alıp birleştirerek stok takibi yapar.

## Özellikler
- API'den veri çekme
- CSV dosyası ile veri birleştirme
- SQLite veritabanına kayıt
- SQL sorguları çalıştırma
- Excel raporu oluşturma

## Kullanılan Teknolojiler
- Python
- Pandas
- Requests
- SQLite
- Openpyxl

## Çalıştırma
python3 main.py
